    工具准备

    1. Python3
    2. pip3 // Python 和 pip 安装不过多赘述，网络上资料也很齐全。
    3. pyxelate // 将图像转换为 8 位像素艺术，具有调色板传输支持和增强的抖动。
    1. scikit-image //安装 skimage 会报错。
